Features

Everything you need to analyse your pace

All the data the game generates, presented in a clean browser dashboard — on any device on your local network.

▶

Auto-capture

Lap times are recorded the instant you cross the finish line. No button presses, no setup per session.

⏱

Sector splits

S1, S2, and S3 times per lap so you can pinpoint exactly where you're losing time.

★

Best lap tracking

Personal best is highlighted in gold. Every other lap shows a delta so the gap is always visible at a glance.

🚗

Tyre compound tracking

Soft, Medium, Hard, Inter, and Wet — each lap records the compound with colour-coded pills in the table.

💾

Persistent storage

Every session and lap is automatically saved to a local SQLite database. Data survives restarts.

📄

CSV export

Download the current session or any past session as a CSV to analyse in Excel, Python, or anything else.

🏆

All-time personal bests

Track PBs are stored per track and session type — a Monaco Qualifying PB and a Monaco Race PB are kept separately.

⛔

Invalid lap flagging

Track-limits violations are marked clearly so a dirty lap never gets mistaken for a real time.

🌐

Works on any device

The dashboard runs in any browser. Open it on your laptop, tablet, or phone — as long as it's on the same network.

How it works

Up and running in three steps

Pitwall IQ uses F1 25's built-in UDP telemetry — no mods or extra software required.

01

Enable UDP in-game

Go to Settings → Telemetry Settings. Set UDP Telemetry to On, IP to 127.0.0.1, and port to 20777. One-time setup.

02

Run the tracker

Requires Python 3.8+ and nothing else — no third-party packages. Uses only the standard library.

python F1_lap_tracker.py
03

Open the dashboard

Navigate to http://localhost:5000 in any browser. Once you enter a session the status switches to LIVE TELEMETRY.

04

Drive normally

Every lap is captured automatically. Browse past sessions, export CSVs, and track your PBs without touching a button.
